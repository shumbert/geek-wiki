# Opensmtpd                                                  
Tables with file back-end need to be manually reloaded:
```                                                    
doas smtpctl update table <table>                      
```                                                    
                                                       
Other funky commands:            
```                              
doas smtpctl log brief    
doas smtpctl log verbose
doas smtpctl monitor    
doas smtpctl show queue   
```
